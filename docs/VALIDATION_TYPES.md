# Validation Types

Supported types include:

- String and pattern types: `email`, `phone`, `url`, `slug`, `uuid`, `ip`, `json`, `domain`, `credit_card`, `mac_address`
- Numeric types: `integer`, `decimal`, `percentage`, `currency`
- Date/time types: `date`, `time`, `datetime`, `timestamp`
- File types: `image`, `file`, `document`, `video`, `audio`
- Structural types: `array`, nested fields via Laravel wildcard syntax
